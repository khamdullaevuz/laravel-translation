## Installation

You can install the package via composer:

```bash
composer require khamdullaevuz/laravel-translation
```

Add the service provider to your `config/app.php`:

```php
// config/app.php
'providers' => [
    ...
    Khamdullaevuz\LaravelTranslation\TranslationServiceProvider::class,
];
```

You can run the migrations with:

```bash
php artisan migrate
```

## Usage

Model

```php
<?php

namespace App\Models;

use Illuminate\Database\Eloquent\Factories\HasFactory;
use Illuminate\Database\Eloquent\Model;
use Khamdullaevuz\LaravelTranslation\Facades\Translation;

class Product extends Model
{
    use HasFactory, Translation;

    protected $fillable = [
        'name',
        'amount',
    ];

    protected $translatable = [
        'name',
    ];
}
```

Create
```php

$product = Product::create([
    'name' => 'Product 1',
    'amount' => 100,
]);

$product->translations()->create([
    'table_name' => 'products',
    'column_name' => 'name',
    'value' => 'Mahsulot 1',
    'locale' => 'uz',
    'foreign_key' => $product->id,
]);
```

Get
```php
$product = Product::find(1);

app()->setLocale('uz');
$product->getAttribute('name');
```
